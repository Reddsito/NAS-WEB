Topología de Red - Servidores

NAS

Almacenamiento conectado en red para guardar y compartir archivos en la red local.

Ruta: 192.168.1.100

Puertos: 80, 443, 21

Servicios: FTP, HTTP, HTTPS

Multimedia

Servidor para streaming de contenido multimedia como videos, música y fotos.

Ruta: 192.168.1.101

Puertos: 80, 443, 1935

Servicios: HTTP, HTTPS, RTMP

DNS

Sistema de nombres de dominio que traduce nombres de dominio a direcciones IP.

Ruta: 192.168.1.102

Puertos: 53

Servicios: DNS

DHCP

Asigna automáticamente direcciones IP a dispositivos en la red.

Ruta: 192.168.1.103

Puertos: 67, 68

Servicios: DHCP
